# 🎨 Color Changer Web App

A simple and interactive Color Changer Web Application built using HTML, CSS, and JavaScript. This project allows users to dynamically change colors and explore UI interactions in a fun way.

---

## 🚀 Live Demo

👉 https://color-changer-web-app.netlify.app/

---

## 📌 Features

- 🎨 Dynamic color changing
- ⚡ Fast and lightweight
- 📱 Fully responsive design
- 💡 Clean and beginner-friendly UI
- 🔄 Instant visual feedback

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
